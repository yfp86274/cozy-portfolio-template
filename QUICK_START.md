# 🚀 快速入門指南

歡迎使用手作人網站模板！這份指南會幫助您在 **5 分鐘內** 讓網站上線。

---

## 🎯 三種使用方式

### 方式一：完全不碰程式碼（推薦新手 👋）

1. **點擊 "Open in Codespaces"** 按鈕
2. 等待環境準備好（約 1-2 分鐘）
3. 當您看到網站預覽視窗時，點擊 **「🎨 啟動 CMS 管理介面」** 任務
4. 在 CMS 介面中填寫您的資料
5. 按下「發佈」即可！

### 方式二：直接編輯設定檔

1. 打開 `site.config.json`
2. 修改裡面的文字（名字、介紹等）
3. 存檔後網站會自動更新！

### 方式三：進階自訂

如果您懂一些程式碼，可以：

- 修改 `src/` 資料夾中的 Vue 組件
- 調整 `tailwind.config.js` 的樣式設定
- 自訂更多功能

## 🎨 讓網站瞬間變漂亮

只要在設定中加入您的「職業」，網站就會自動套用最適合的風格！

```json
{
  "profile": {
    "name": "您的名字",
    "profession": "knitter"  // 👈 加上這行！
  }
}
```

### 可用的職業風格

| 類別    | 職業代碼                                                          | 說明             |
|-------|---------------------------------------------------------------|----------------|
| 🍳 餐飲 | `chef`, `baker`, `barista`                                    | 廚師、烘焙師、咖啡師     |
| 🧶 手作 | `knitter`, `potter`, `jeweler`, `leatherworker`, `woodworker` | 編織、陶藝、珠寶、皮革、木工 |
| 🎨 藝術 | `artist`, `illustrator`, `photographer`, `designer`           | 藝術家、插畫、攝影、設計   |
| 🌸 花藝 | `florist`, `gardener`                                         | 花藝師、園藝師        |
| 💆 療癒 | `therapist`, `yoga`                                           | 治療師、瑜伽老師       |
| 🏠 空間 | `architect`, `interior`                                       | 建築師、室內設計       |
| 🎵 音樂 | `musician`                                                    | 音樂人            |
| 📝 文字 | `writer`, `teacher`                                           | 作家、老師          |

## 🖼️ 上傳您的圖片

1. 將圖片放入 `public/images/` 資料夾
2. 在設定中使用 `/images/您的圖片.jpg` 作為路徑

### 圖片建議

| 圖片類型  | 建議尺寸        | 建議大小     |
|-------|-------------|----------|
| 大頭照   | 400×400 px  | < 500 KB |
| 作品圖   | 1200×800 px | < 2 MB   |
| 社群分享圖 | 1200×630 px | < 1 MB   |

💡 **小提示**：可以使用 [TinyPNG](https://tinypng.com) 壓縮圖片！

## 📤 讓網站上線

### 使用 GitHub Pages（免費！）

1. 確認您的程式碼已經推送到 GitHub
2. 到 Repository 的 Settings → Pages
3. Source 選擇 "GitHub Actions"
4. 每次您更新 `site.config.json`，網站就會自動更新！

## ❓ 遇到問題？

### 🔍 自我檢查

在終端機執行：

```bash
npm run validate
```

這會告訴您設定檔有什麼問題！

### 📋 常見問題

**Q: 網站一片空白？**

- 檢查 `site.config.json` 是否有語法錯誤
- 執行 `npm run validate` 查看詳細說明

**Q: 圖片顯示不出來？**

- 確認圖片放在 `public/images/` 資料夾
- 檢查檔案名稱是否完全正確（包括大小寫）

**Q: 顏色沒有套用？**

- 顏色格式應該是 `#RRGGBB`，例如 `#FF5733`
- 如果設定了職業，顏色會自動套用職業預設

### 🆘 還是不行？

1. 查看 `CONFIG_GUIDE.md` 完整設定指南
2. 到 GitHub Issues 發問，我們會幫助您！
