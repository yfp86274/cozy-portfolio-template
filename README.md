## 🚀 四步驟建立你的網站

```
┌─────────┐     ┌─────────┐     ┌─────────┐     ┌─────────┐
│         │     │         │     │         │     │         │
│  1.Fork │ ──▶ │ 2.設定  │ ──▶ │ 3.傳圖  │ ──▶ │ 4.完成! │
│         │     │         │     │         │     │         │
└─────────┘     └─────────┘     └─────────┘     └─────────┘
   複製專案       填表單複製       拖拉上傳         等1分鐘
```

---

### 步驟 1️⃣ Fork（複製）這個專案

1. 點擊頁面右上角的 **Fork** 按鈕
2. 等幾秒鐘，你就有自己的網站倉庫了！

![Fork 按鈕位置](https://docs.github.com/assets/images/help/repository/fork_button.jpg)

---

### 步驟 2️⃣ 用「設定精靈」生成你的設定

**這是最簡單的方式！**

1. 部署完成後，打開：`https://你的用戶名.github.io/你的倉庫名/config-maker.html`
2. 選擇你的職業（網站會自動變成最適合的風格）
3. 填入你的名字、Email、社群連結
4. 點擊「**複製設定代碼**」按鈕

> 💡 **還沒部署？** 可以先用 [GitHub 的原始檔預覽](./public/config-maker.html) 或下載 `public/config-maker.html` 到電腦用瀏覽器打開

---

### 步驟 3️⃣ 把設定貼到 GitHub

**🪄 魔法技巧：按「.」鍵開啟網頁編輯器**

1. 回到你的 GitHub 倉庫頁面
2. 按鍵盤上的 **`.`** 鍵（驚嘆號旁邊那個點）
3. 等幾秒，會打開一個網頁版的編輯器
4. 在左側找到 `site.config.json` 並點開
5. 全選（`Ctrl+A` 或 `Cmd+A`）→ 貼上（`Ctrl+V` 或 `Cmd+V`）
6. 按 `Ctrl+S` 或 `Cmd+S` 儲存

> ⚡ **為什麼用網頁編輯器？** 因為可以直接拖拉上傳圖片！比下載軟體方便多了。

---

### 步驟 4️⃣ 上傳你的作品照片

**在網頁編輯器裡操作：**

1. 在左側找到 `src` → `assets` → `works` 文件夾
2. 右鍵點擊 `works` → 新建資料夾
3. 命名你的作品，例如：`01_我的第一個作品`
4. **直接把照片拖進去！**

**文件夾命名小技巧：**

```
works/
├── 01_冬日圍巾/          ← 數字開頭，會排在最前面
│   ├── cover.jpg         ← 這張會當封面（也可以不放，系統會自動選第一張）
│   ├── 細節1.jpg
│   └── 細節2.jpg
├── 02_手工皂/
│   └── ...
└── 我的貓咪/             ← 不加數字也可以！會按字母排序
    └── photo.jpg
```

> 📝 **想加作品描述？** 在作品文件夾裡創建 `readme.md`，直接寫文字就好

---

### 🎉 完成！

1. 儲存所有變更（網頁編輯器左邊有個勾勾圖示）
2. 等待約 1-2 分鐘（GitHub 正在幫你建置網站）
3. 打開：`https://你的用戶名.github.io/你的倉庫名/`
4. **你的作品集網站上線了！**

## ❓ 常見問題

### 網站沒有更新？

- 去倉庫的 **Actions** 頁面看有沒有綠色勾勾 ✓
- 清除瀏覽器緩存：`Ctrl+Shift+R`（Mac 是 `Cmd+Shift+R`）

### 我打錯字了，設定檔壞掉了？

**別擔心！** 這個模板有自動修復功能。常見的錯誤（多了逗號、少了引號）都會在建置時自動修正。

如果真的壞掉了，用設定精靈重新生成一份就好。

### 圖片沒有顯示？

- 確認是圖片格式：`.jpg`、`.png`、`.webp`、`.gif`
- 檔名避免特殊符號，用中文英文數字都可以

### 想改顏色？

設定精靈會幫你選好最適合的配色，但如果想自己調整，可以在 `site.config.json` 加上：

```json
{
  "profile": {
    ...
  },
  "theme": {
    "primaryColor": "#8B4513",
    "backgroundColor": "#FDF5E6"
  }
}
```
